¿Porque me sale error 404?

Indica un error interno, Puede deberse a un error en la pagina porto que el servidor no puede responder

El sistema no detecta mi rostro

Probablemente sea por un error en el registro de la foto, revisa que la foto no tenga un mal enfoque o algun accesorio que no permita que tu rostro sea visible

¿Porque me sale error 500?

Indica un error interno, Puede deberse a un error en la pagina porto que el servidor no puede responder

¿Porque me sale error 504?

Indica que el tiempo de espera para cargar la página web se ha agotado. Puede estar generado por un error en el propio servidor,o porque nuestra página web contiene algún error por Io que no puede cargarse.
